# 11172: Relational Operator
![11172: Relational Operator]()

## Solution
```C++
#include<iostream>

using namespace std;

int main(void)
{
	int n;
	long long int a, b;
	
	while(cin >> n)
	{
		for(int i = 0; i < n; i++)
		{
			cin >> a >> b;
			
			if(a > b)
				cout << ">" << endl;
			else if(a < b)
				cout << "<" << endl;
			else
				cout << "=" << endl;
		}
	}
	
	return 0;
}
```
